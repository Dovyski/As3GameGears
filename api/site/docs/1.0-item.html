Gets information about an specific item. An item is indexed by a slug, which is the name of the item with spaces and punctuation signs replaced with - (hyphen).

Parameters

Name Type Description
name string The slug of the item. E.g. flixel.
props string A coma separated string containing the name fields of the item you want. E.g.: if you want the API to return just the name and description of an item, props would be name,description.

Return

Requesting http://api.as3gamegears.com/1.0/item/flixel will produce the following return:

  "name": "Flixel",

  "sample": "package { import org.flixel.*; (...)",
  "license": [
    {
      "id": 46,

    }
  ],
