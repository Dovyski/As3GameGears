Lists the licenses used by the tools in the site. You can get information about an specific license using its slug, which is the name of the license with spaces and punctuation signs replaced with - (hyphen). If no slug is provided, the method returns all known licenses.

Parameters

Name Type Description
slug string The slug of the license. E.g. simplified-bsd.

Return

Requesting http://api.as3gamegears.com/1.0/licenses/simplified-bsd will produce the following return:


{
  "id": 49,
  "name": "Simplified BSD",
  "slug": "simplified-bsd"
}
